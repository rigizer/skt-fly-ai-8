# 가상환경 설정
- 가상환경 생성
```
python -m venv .venv
```

- 가상환경 접속
```
.\.venv\Scripts\activate
```

- 패키지 일괄 설치
```
pip install -r requirements.txt
```

- 패키지 일괄 내보내기
```
pip freeze > requirements.txt
```

# 실행
```
streamlit run 01.hello_streamlit.py
```
